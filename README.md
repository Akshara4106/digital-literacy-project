# Digital Literacy Project – CSE0001

## 📌 Project Overview
This project is part of the Digital Literacy (CSE0001) course at VIT Bhopal University. It focuses on developing essential digital skills required to use online platforms safely, responsibly, and professionally.

The project includes creating an infographic, building a digital portfolio, exploring coding platforms, practicing email etiquette, and understanding cybercrime awareness.

---

## 🎯 Objectives
- To understand digital literacy and its importance  
- To promote safe internet practices  
- To build a professional online presence  
- To learn coding and collaboration tools  
- To understand cybercrime and prevention  

---

## 📂 Project Structure

### Task 1 – Infographic
Created a digital literacy awareness infographic using Canva.

### Task 2 – Digital Portfolio
Built professional profiles on GitHub and LinkedIn.

### Task 3 – Platforms
Solved a coding problem on HackerRank and created a Google Form quiz.

### Task 4 – Email Etiquette
Drafted professional emails and created a social media checklist.

### Task 5 – Cybercrime
Prepared a case study and prevention checklist on phishing.
